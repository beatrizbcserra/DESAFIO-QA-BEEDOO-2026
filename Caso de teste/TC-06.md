# TC-06: Formato de URL Inválido.

**Objetivo:** Ver se o usuário consegue realizar o cadastro de curso com formato de URL de imagem inválida.
- **Pré-condição:**
  - Estar na página inicial da aplicação.

- **Passos:**
  1. Clicar no link "Cadastrar Curso".
  2. Preencher o campo "Nome do curso" com dados válidos (Ex. QA Engineer).
  3. Preencher o campo "Descrição do curso" com dados válidos (Ex. O curso foca em garantir a qualidade de softwares através de testes manuais e automatizados, abrangendo ferramentas como Cypress, Selenium,
     APIs, metodologias ágeis (Scrum) e Git).
  4. Preencher o campo "Instrutor" com dados válidos (Ex. Maria).
  5. Preencher o campo "Url da imagem de capa" (opcional) com um texto comum (Ex: "imagem_teste").
  6. Selecionar, respectivamente, nos campos "Data de ínicio" e "Data de fim" a conclusão e ínicio do curso.
  7. Preencher o campo "Número de vagas" com dados válidos.
  8. Selecionar no campo "Tipo de curso" o formato de ensino (Ex. Online)
  9. Preencher o campo "Link de inscrição" com dados válidos.
  10. Clicar no botão [CADASTRAR CURSO].

- **Resultado Esperado:**
  - O usuário não consegue realizar o cadastro com sucesso.
  - Uma mensagem de erro é exibida (Ex. Cadastro de curso não realizado, favor inserir as datas de ínicio e conclusão corretas).
